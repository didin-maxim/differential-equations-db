## Добавлено

Добавлены только задачи с новым переносимым методом и без явного дубля в насыщенных кластерах:

- `msu-ode-2004-2-delay-step-value`: метод шагов для уравнения с запаздыванием.
- `msu-ode-2009-2-prescribed-zero-sequence`: обратное построение гладкого коэффициента по заданным нулям.
- `msu-ode-2014-1-no-entire-abs-linear`: отсутствие решения на всей прямой через максимальное продолжение.
- `msu-ode-2019-3-nonzero-field-nonunique-plane`: ненулевое непрерывное поле на плоскости с неединственностью.
- `msu-ode-2022-2-oscillatory-semistability`: устойчивость при накоплении равновесий около нуля.

## Отсеяно

Повторно не импортировались задачи на уже насыщенные или близко насыщенные сюжеты: матричная экспонента без существенного ДУ-содержания, стандартный Риккати, интегрирующие множители, штурмовские нули без нового хода, однотипная фазовая классификация и вычислительно тяжелые линейные системы.

Некоторые хорошие задачи старого архива оставлены для будущего точечного разбора, если понадобится усилить конкретный дефицит: 2016/2 о конечном конце непродолжаемого решения, 2017/1 о глобальности по последовательности начальных данных, 2018/2 о спиральном сжатии и 2020/1 с рисунком фазового портрета.

## Симуляция экзамена

Создан proposal `data/exam_simulation/question_batches/msu-ode-archive-old.yaml`. В него вошли 4 вопроса: один numeric, два multiple-choice и один self-check. Карточка 2009/2 не предложена в режим, потому что качественная конструкция плохо автопроверяется без превращения в слишком раскрывающую подсказку.
