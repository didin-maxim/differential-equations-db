# Разбиение файлов кластеров

Дата: 2026-05-27.

`data/task_clusters/clusters.yaml` разбит на тематические JSON-compatible YAML файлы. Старый файл оставлен как нейтральный контейнер:

```json
{
  "task_clusters_version": 1,
  "clusters": []
}
```

Карта файлов:

- `first_order.yaml`: уравнения первого порядка, характеристики ЧП, интегрирующие множители, Риккати/Бернулли, восстановление ОДУ, неразрешенные ОДУ, ортогональные траектории, угадывание решения задачи Коши.
- `linear.yaml`: скалярные линейные ОДУ, вариация постоянных, переменные коэффициенты, Флоке, степенные ряды, вронскианные преобразования.
- `systems.yaml`: линейные системы, матричная экспонента, фундаментальная матрица, первые интегралы плоских систем.
- `qualitative.yaml`: фазовая прямая, существование/единственность/продолжение, энергетические оценки, предельные циклы.
- `bvp.yaml`: спектральные и краевые задачи, Штурм, функции Грина, резонанс.
- `variational.yaml`: вариационное исчисление и параметрическая зависимость через вариационное уравнение.
- `olympiad.yaml`: олимпиадные кластеры.
- `misc.yaml`: оставшиеся межтематические кластеры.

`tools/build_index.py` теперь, как и `tools/check_clusters.py`, читает все `data/task_clusters/*.yaml`.
