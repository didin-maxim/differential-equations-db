# Обзор архива IMC на кандидаты по дифференциальным уравнениям

Область обзора: International Mathematics Competition for University Students (IMC), официальный архив <https://www.imc-math.org.uk/>. PDF с условиями и решениями проверялись по годам на ОДУ, дифференциальные неравенства, интегрирующие множители, линейные дифференциальные операторы и кандидаты на матричную экспоненту/системы. Карточки этим обзором не добавлялись.

## Проверенные архивы и годы

- Официальные страницы архива IMC: <https://www.imc-math.org.uk/?year=YYYY&item=problems>.
- Проверены годы с 1994 по 2025 включительно.
- Проверены PDF, реально связанные с этими страницами: всего 89 официальных PDF. В старых годах часто встречаются объединенные PDF "Problems and Solutions"; в новых обычно отдельные PDF по дням, задачам и решениям.
- Поиск велся по словам `differential`, `derivative`, `differentiable`, `f'`, `f′`, обозначениям старших производных, `matrix exponential`, `ODE`, `d/dt` и близким линейным/системным словам. Найденные кандидаты затем сверялись с условиями и решениями.

## Пробелы и оговорки

- IMC 2026 was not checked for problems: as of 2026-05-26 the event is scheduled for 2026-07-27 to 2026-08-03, so the competition problems are not yet an archive item.
- I did not add cards; this file is only a review/scouting report.
- I did not find credible IMC candidates involving matrix exponentials or explicit systems of ODEs in 1994-2025. The matrix problems found in the archive were algebraic/linear algebra rather than differential systems.
- Because the first archive pass used extracted PDF text, a purely image-encoded formula could theoretically evade keyword search. The official PDFs I checked produced usable text for the relevant years, and the solution review covered all keyword hits that looked near the inclusion criteria.
