# Локальный файл 8.pdf: программа и теоретические задачи

Источник: `C:\Users\Admin\Desktop\ДУ\8.pdf`.

Файл содержит утвержденную 16 января 2025 г. программу дисциплины "Дифференциальные уравнения" для 2 курса, 4 семестра: 30 часов лекций, 30 часов практических занятий, экзамен в 4 семестре. Составители: А. М. Бишаев, В. Ю. Дубинская, С. Е. Жуковский, С. Д. Животов, А. Ю. Семенов. Задания составлены С. Д. Животовым.

## Темы программы

1. Основные понятия и простейшие типы ОДУ первого порядка: разделяющиеся переменные, однородные, линейные, полные дифференциалы, интегрирующий множитель, уравнения Бернулли и Риккати, уравнения первого порядка, не разрешенные относительно производной, понижение порядка.
2. Задача Коши: существование, единственность, продолжение, зависимость от параметров и начальных данных, особые решения.
3. Линейные уравнения и системы с постоянными коэффициентами: общее решение, квазимногочлены, уравнение Эйлера, жорданова форма, матричная экспонента, преобразование Лапласа по усмотрению лектора, краевые задачи с малым параметром по усмотрению лектора.
4. Линейные уравнения и системы с переменными коэффициентами: фундаментальная система и матрица, структура общего решения, определитель Вронского, формула Лиувилля-Остроградского, вариация постоянных, формула Коши, теорема Штурма и следствия, уравнение Бесселя и асимптотика по усмотрению лектора.
5. Автономные системы: фазовые траектории, классификация равновесий линейных систем второго порядка, локальное поведение нелинейных систем, выпрямление траекторий по усмотрению лектора, устойчивость и асимптотическая устойчивость, фазовый объем, формула Лиувилля, теорема Пуанкаре без доказательства.
6. Первые интегралы автономных систем и линейные однородные уравнения в частных производных первого порядка: критерий первого интеграла, число независимых первых интегралов, задача Коши для линейного уравнения первого порядка в частных производных.
7. Элементы вариационного исчисления: простейшая задача, свободные концы, несколько неизвестных функций, производные высших порядков, изопериметрическая задача и задача Лагранжа без доказательства.

Первое задание, срок 15-21 марта:

## Импортированные теоретические задачи

В карточки перенесены самодостаточные задачи первого задания:

- T1: ограниченные решения уравнения Бесселя около нуля и вронскиан;
- T2: энергетическая оценка для `y'' + exp(2/(x+1)) y = 0`;
- T3: краевая задача `y'' + q(x)y = 0`, `q(x) <= 0`, решена через теорему сравнения Штурма;
- T4: не более трех нулей у `y'' + 2x^2 y' + (2x+1)y = 0`;
- T5: существование точки с `y'(xi)=0` для `y'' + (2+cos 3x)y = 0`;
- T6a: бесконечно много нулей у нетривиального решения уравнения Бесселя на `(0,+infty)`;
- T6b: расстояние между последовательными нулями решения уравнения Бесселя стремится к `pi`.

Второе задание содержит в основном вычислительные задачи по первым интегралам, линейным PDE и вариационному исчислению; они не переносились в эту теоретическую пачку.

## Граница курса

`standard_course_methods`:

- формула Лиувилля-Остроградского для вронскиана;
- энергетическая монотонность для линейного уравнения второго порядка;
- существование и единственность задачи Коши для линейного уравнения;
- теорема сравнения Штурма в обычной форме;
- приведение линейного уравнения второго порядка к нормальной форме без первой производной;
- простые следствия Ролля для нулей и критических точек.

`advanced_standard_course`:

- применение сравнения Штурма не только к явному синусу, но и к запрету второй краевой точки при `q <= 0`;
- подсчет числа нулей после замены Лиувилля и разбиения оси на области разных знаков коэффициента;
- осцилляция уравнения Бесселя через асимптотическое сравнение с постоянным коэффициентом;
- оценка промежутков между нулями через угол Прюфера при `Q(x) -> 1`.

`beyond_standard_course`:

- внешняя теория специальных функций Бесселя, асимптотические формулы Бесселя и глубокая спектральная теория не требуются для импортированных решений;
- задача T6b использует угол Прюфера, который может быть продвинутым для базового курса, но доказательство в карточке самодостаточно и опирается только на элементарное дифференцирование.
