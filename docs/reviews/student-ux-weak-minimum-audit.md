# UX-аудит: слабый студент, письменный минимум

Дата: 2026-05-27.

Роль: слабый студент МФТИ, задача - быстро найти простые неолимпиадные задачи для закрытия минимума/пересдачи по дифференциальным уравнениям.

## Что проверялось

- Открытие viewer как пользователь через локальную сборку `http://127.0.0.1:4177/viewer/`.
- Главная страница: счетчики, русские подписи, быстрые режимы.
- Фильтры: `Письменный минимум`, `без олимпиадных`, диапазон `простые: идея/техника <= 4`.
- Несколько карточек уровня `idea_score`/`technical_score` 2-4.
- Скрытие идей и решений по умолчанию.
- Открытие идей/решений.
- Отображение TeX и узкой типографики на маркере `oral-exam-strong-10-sturm-comparison-one-zero`.

## Наблюдения

- Главные счетчики понятны: `357 карточек`, `294 задач`, `521 связей`, `32 кластеров`.
- Подписи фильтров русские: `Идейная сложность`, `Техническая сложность`, `Олимпиадность`, `Источник`, `Кластер`, `Идея`.
- Кнопки быстрых режимов читаются по-русски. Для слабого студента полезнее всего `Письменный минимум` и фильтр `без олимпиадных`.
- Фильтр `Письменный минимум` автоматически включает исключение олимпиадных карточек, что хорошо для пересдачи.
- Идеи и решения закрыты по умолчанию: в проверенных карточках `details[open] = 0`.
- После открытия подсказки и решение показываются нормально, без случайного раскрытия соседних карточек.
- Маркер TeX отображается корректно: видны `[a,a+π]`, `(a,a+π]`, `q(x)≥1`; сырого `[a,a+pi]` и `>=` не видно.

## Исправление

Найден один UX-недочет: быстрый режим `Письменный минимум` сортировал карточки только по технической сложности. Поэтому первыми появлялись задачи с очень малой техникой, но с идейной сложностью 5, что для слабого студента хуже, чем задачи `идея 2-3`.

Исправлено в `tools/build_viewer.py`:

- добавлена сортировка `Идея и техника ↑`;
- быстрый режим `Письменный минимум` теперь включает эту сортировку.

После правки первые карточки режима `Письменный минимум`:

- `Простая задача Коши с разделяющимися переменными` (`идея 2`, `техника 2`);
- `Простое неоднородное уравнение с постоянными коэффициентами` (`идея 2`, `техника 2`);
- `Система 2x2 с диагональной матрицей` (`идея 2`, `техника 2`);
- `Энергия для простого осциллятора` (`идея 2`, `техника 2`);
- далее идут задачи уровня `идея 3`, `техника 2`.

## Спорные моменты

- Кнопка `Кластеры` показывает количество карточек в кластерном режиме, а не число кластеров. Это не баг, но для первого пользователя может быть неочевидно, потому что рядом в шапке есть `32 кластеров`.
- В карточках видны технические теги вроде `oral_exam`, `standard_course_methods`, `mipt_core`. Для подготовки это полезно, но слабому студенту может быть шумно. Сейчас не правил: это общий дизайн решения с метаданными.
