# Reviews

Здесь лежат рабочие отчеты по источникам перед импортом карточек в `data/`.

## Активные проходы

- `olympiad-selection-criteria.md` - критерии отбора задач по дифференциальным уравнениям и дифференциальным неравенствам.
- `olympiad-scan-putnam.md` - проход по архивам Putnam.
- `olympiad-scan-imc.md` - проход по архивам IMC.
- `olympiad-scan-ru-student.md` - проход по русскоязычным студенческим олимпиадам.
- `olympiad-scan-other-international.md` - проход по прочим международным студенческим олимпиадам.

Файлы `olympiad-scan-*.md` являются очередью кандидатов, а не источником истины
для viewer. В `data/problems/` переносятся только задачи с проверенным условием,
ясным механизмом решения и подходящими тегами.
