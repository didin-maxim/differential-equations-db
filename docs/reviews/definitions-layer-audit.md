# Аудит слоя определений

Дата: 2026-05-27.

## Что добавлено

В `data/definitions/definitions.yaml` добавлен компактный слой из 20 определений:

- общее решение, частное решение, интегральная кривая;
- особое решение и дискриминантная кривая неявного уравнения;
- линейная независимость решений и фундаментальная система решений;
- устойчивость по Ляпунову, асимптотическая устойчивость, фазовая траектория;
- собственное значение краевой задачи;
- ортогональные траектории;
- характеристика уравнения первого порядка в частных производных;
- функционал, экстремаль, вариация, условие трансверсальности;
- изопериметрическая задача, сопряженная точка, условие Якоби.

Существующие определения задачи Коши, точного уравнения, интегрирующего множителя, вронскиана, матричной экспоненты, краевой задачи, положения равновесия и фазовой прямой не дублировались. Их формулировки оставлены без языковой правки, чтобы не конфликтовать с параллельным редакторским проходом.

## Связи с задачами

Связи задача -> определение добавлены через существующий механизм `definition_ids` в формулировках задач. Отдельные `data/relations/relations.d/*.yaml` не использовались, потому что текущие валидаторы разрешают endpoints relations только между карточками из `data/problems`.

Добавлено примерно 123 новых ссылки на определения. Основные зоны:

- характеристики ЧП первого порядка: 10 ссылок;
- вариационное исчисление: функционал, экстремаль и вариация по 15 ссылок, трансверсальность 3 ссылки, изопериметрические задачи 2 ссылки, Якоби/сопряженные точки по 2 ссылки;
- устойчивость и фазовая геометрия: устойчивость по Ляпунову 12 ссылок, асимптотическая устойчивость 7 ссылок, фазовая траектория 3 ссылки;
- неявные уравнения: дискриминантная кривая 4 ссылки, интегральная кривая 4 ссылки, особое решение 3 ссылки;
- ортогональные траектории: 3 ссылки;
- краевые собственные значения: 3 ссылки;
- линейная теория: общее/частное решение, ФСР и линейная независимость добавлены к выбранным задачам на вариацию постоянных, вронскиан и восстановление общего решения.

## Оставлено на следующий проход

- отдельные определения для предельного цикла, устойчивости цикла и отображения Пуанкаре;
- монодромия, мультипликаторы Флоке и связь с фундаментальной матрицей;
- условие Лежандра как отдельная definition-карточка рядом с уже добавленным условием Якоби;
- вариационное уравнение по параметру и производная потока по начальным данным;
- формула Лиувилля и determinant/trace-интерпретация для фундаментальной матрицы;
- более тонкое разделение характеристик линейных, квазилинейных и полулинейных ЧП, если база будет расширять PDE-блок.
