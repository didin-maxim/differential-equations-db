# Аудит кластера: угадать решение задачи Коши и применить единственность

Дата: 2026-05-27.

## Цель

Новый кластер `guess-cauchy-solution-uniqueness` отделяет прикладной прием от широкого `existence-uniqueness-continuation`. Здесь задача не в доказательстве теоремы, не в продолжении решения и не в контрпримерах к Липшицу, а в коротком ходе:

1. угадать конкретное решение по начальным данным;
2. проверить подстановку;
3. проверить локальную единственность;
4. заключить, что найденное решение является единственным решением задачи Коши.

## Точные попадания из базы

- `one-dimensional-autonomous-no-crossing`: постоянное равновесное решение и единственность запрещают пересечение равновесия.
- `two-solutions-touching`: если два решения имеют одинаковые данные Коши в одной точке, они совпадают на общем интервале.
- `oral-exam-excellent-nonautonomous-sign-barrier`: нулевое решение и локальная единственность дают знаковый барьер.
- `cluster-linear-variable-graph-direct-tangency`: две разные функции не могут быть решениями одного линейного ОДУ при одинаковых y(x0), y'(x0).
- `waterloo-fundamental-matrix-flow-inverse`: потоковая форма единственности для линейной автономной системы; включена как близкий представитель, но не как чистая задача на угадывание.

Не включались как точные попадания: `resit-pass-3-cauchy-uniqueness-check` и `weak-pass-nonunique-continuous-counterexample`, потому что их центр - неединственность/неприменимость теоремы; `local-du-programs-other-years-pendulum-sin-phase-portrait`, потому что это полный фазовый портрет, а не конкретная задача Коши.

## Добавленные карточки

- `cluster-guess-cauchy-autonomous-equilibrium`: автономное уравнение `y'=e^(y^2) sin y`, начальные данные в равновесии.
- `cluster-guess-cauchy-pendulum-inverted-equilibrium`: математический маятник `theta''+sin theta=0`, данные `(pi,0)`, без расчета эллиптических интегралов.
- `cluster-guess-cauchy-nonlinear-oscillator-zero`: нелинейное ОДУ второго порядка с нулевыми данными.
- `cluster-guess-cauchy-system-diagonal-invariant`: нелинейная система с инвариантной диагональю `x=y`.
- `cluster-guess-cauchy-designed-straight-line`: неравновесная угаданная траектория `y=t`.

Все новые карточки являются self-authored/model tasks. Это честно отмечено в `sources` и `editorial.notes`.

## Политика насыщения

Кластер уже покрывает основные варианты:

- автономное равновесие;
- нулевое решение уравнения второго порядка;
- классический маятник;
- инвариантная прямая/диагональ системы;
- неравновесная угаданная кривая;
- существующие задачи на совпадение решений по данным Коши.

Дальше не стоит добавлять задачи, которые только меняют гладкую правую часть при том же нулевом ответе. Полезными могут быть только новые механизмы: симметрия решения, проверенный классический источник с формулировкой "не находя общего решения", или трехмерная инвариантная плоскость без дублирования диагонали.
