# UX-аудит: телефон и быстрый поиск перед экзаменом

Дата: 2026-05-27.

Роль: студент, который открывает базу с телефона или узкого экрана и хочет быстро найти карточки перед экзаменом.

## Что проверено

- Главная страница viewer: счетчики, быстрые режимы, фильтры, активные фильтры.
- Фильтры по идейной и технической сложности, источникам, авторам, кластерам, стандартным идеям, тегам и наличию рисунков.
- Карточки: длинные id, теги, бейджи, условия, раскрываемые блоки идей и решений.
- TeX/типографика на маркере `oral-exam-strong-10-sturm-comparison-one-zero`: исходное `[a,a+pi]` в данных преобразуется viewer-типографикой в `[a,a+π]`, а `>=1` в `≥1`.
- Рисунки и SVG: в индексе 10 локальных image-asset, битых локальных путей не найдено.
- Закрытость подсказок: в сгенерированном HTML нет `details.reveal` с атрибутом `open`.

Полный browser-smoke через Playwright в этой сессии не запустился: в локальном Node runtime нет пакета `playwright`. Поэтому проверка выполнена статически по `viewer/index.html`, `index/generated.json` и CSS/JS-инвариантам после локальной сборки.

## Что исправлено

- Усилена мобильная CSS-защита от горизонтального разъезда страницы: `html, body` ограничены по ширине и получают `overflow-x: clip`.
- Для карточек, условий, TeX-контента, блоков решения, чипов и бейджей добавлены переносы длинных слов/id и ограничения по ширине.
- Кнопки быстрых режимов и кнопки `Показать/Скрыть` теперь могут переноситься внутри своей ширины на узком экране.
- Сетка изображений теперь использует безопасный `minmax(min(220px, 100%), 1fr)`, а на экранах до 480px переходит в одну колонку.
- На экранах до 480px уменьшены отступы карточек и страницы, статистика главной страницы становится одноколоночной.

## Статические smoke-проверки

- `html lang="ru"`: OK.
- `viewport` для мобильных экранов: OK.
- CSS media rules `max-width: 860px` и `max-width: 480px`: OK.
- Фильтр кластеров присутствует: OK.
- Подписи `Идейная сложность` и `Техническая сложность` в публичном HTML присутствуют: OK.
- KaTeX и fallback-типографика подключены: OK.
- Локальные image assets: `10`, отсутствующих файлов `0`.
- Счетчики индекса: `357` карточек, `294` задачи, `32` кластера.

## Итог

Viewer стал заметно устойчивее для телефона: основные фильтры и карточки должны помещаться в узкую ширину без случайного горизонтального скролла, длинные формулы и id не распирают карточки, решения и идеи по-прежнему скрыты по умолчанию.
