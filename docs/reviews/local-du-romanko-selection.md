Для текущего импорта использовались дефицитные разделы стандартной программы: §10, §16, §17, §20, §21, §22.

## Принцип отбора

Не импортировались длинные цепочки задач из §19 "Простейшая вариационная задача": этот кластер уже отмечен как потенциально перегружаемый. Вместо этого выбраны разные механизмы:

- короткий отрезок + сравнение Штурма для обратимости краевой задачи;
- первый интеграл, который доказывает центр через замкнутые уровни;
- линейное ЧП первого порядка с двумя инвариантами характеристик;
- вариационная задача с производными высшего порядка;
- вариационный функционал от двух неизвестных функций;
- две изопериметрические задачи: линейная связь и L2-нормировка;
- достаточное условие Лежандра-Якоби для строгого слабого максимума.
