## Отсев дублей и технических задач

`DE2006-07.pdf` содержит много задач на первые ОДУ, линейные уравнения, линейные системы и разностные уравнения. Внутри базы уже есть представители этих механизмов: интегрирующие множители, Бернулли/Риккати, Клеро/Лагранж, линейные системы с жордановой структурой, переменные коэффициенты и уравнения с неразрешенной производной. Новые карточки из этого PDF не добавлены: большинство задач отличаются только коэффициентами или дают тяжелый вычислительный вариант.

`2009.pdf` просмотрен визуально: текстовый слой почти отсутствует. Варианты 91-94 имеют общий шаблон: постоянные коэффициенты, переменные коэффициенты, квадратичные вариационные функционалы, линеаризация равновесий, Клеро/Лагранж, PDE-характеристики, матричная экспонента 2x2 и последняя качественная задача на ограниченность. Добавлены только один PDE-представитель и одна качественная задача на ограниченность. Варианты `y''+e^x y=0`, `y''+e^(2x)y=0`, `y''+x^2y=0` не добавлены как близкие по плану; первый почти совпадает с уже имеющейся MSU-карточкой.

`DE2010.pdf` содержит несколько вычислительных вариантов, уже покрытых кластерами: системы с постоянными коэффициентами, линейные уравнения, линеаризация равновесия, вариационные квадратичные функционалы, неразрешенные ОДУ и характеристики. Добавлена задача про `sin A`, потому что это компактная проверка понимания матричной экспоненты без вычислительного монстра.

## Симуляция экзамена

Большинство письменных задач не перенесены напрямую в симуляцию: они слишком технические или в интерактиве дают крупную подсказку. Предложены 4 устных/интерактивных версии:

- первый интеграл характеристик для PDE;
- численное значение `u(1,1)` после решения PDE;
- формула для `det(sin A)`;
- self-check на доказательство ограниченности `y''+xy=0`.

Эти вопросы пока лежат в proposal-файле и не подключены к основному `questions.yaml`.
