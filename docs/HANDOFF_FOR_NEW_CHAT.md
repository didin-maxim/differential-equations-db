# Handoff

Для продолжения работы начните с:

```powershell
git status --short
python tools\check_encoding.py
python tools\validate.py
python tools\audit_rules.py --max-items 40
python tools\search.py query "ключевое слово"
```

Главные направления следующего прохода:

- добавить больше задач по системам, краевым задачам и качественной теории;
- привязать конкретные источники МФТИ/листки/олимпиады вместо общих seed-ссылок;
- расширить связи только после проверки переносимого метода решения;
- при необходимости сделать предметный интерактив для фазовой прямой и поля направлений.
